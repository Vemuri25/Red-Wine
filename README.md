# Wine Deployment (static)

This repository is packaged to be recognized as a static HTML/CSS/CSV project (not a Jupyter Notebook repo).

Files:
- index.html — static viewer with a 100% badge
- styles.css — styling
- winequality-red.csv — dataset (1599 rows × 12 columns)
- .gitattributes — tells GitHub Linguist to ignore notebooks and prefer HTML/CSS for language stats
- .nojekyll — prevents Jekyll processing on GitHub Pages
